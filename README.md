cloudbenchmark
---

Benchmark suite for cloud environment

# install

```
$ sudo pip install cloudbenchmark
```

# usage

```
$ cloudbenchmark -j s3-throughput-small -b [RESULT_BUCKET]
```

## job list

- `s3-throughput-small`
  - `--target-s3 [TARGET_BUCKET]`
- `s3-throughput-large`
  - `--target-s3 [TARGET_BUCKET]`
